App Description:
- This web application is a social platform prototype that allows users to create and manage their own accounts. Currently, the system focuses on core identity management and profile customization.

Core Functions:
- Authentication: Secure Sign-in and Sign-out flow (redirects to the landing page).
- User Profile: View personal account information.
- Settings Page: Allows users to modify their personal description and bio.
- Admin Dashboard: A dedicated interface for administrators to create new accounts.
- All data is synchronized directly with a MariaDB/SQL database named 'socialnet'.

Development Status:
- Social Connectivity: The function to connect or follow other accounts is currently under development.

Dynamic Queries: The "Profile" page is designed to handle query statements for 'owner' ID. If the query is blank, it defaults to displaying the currently logged-in user's profile.
